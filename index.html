🗄️ Data Management

📦 How Your Data is Stored

🔒
IndexedDB (Local) — FinTrack stores all data in your browser's IndexedDB. No server, no account, no data leaves your device.
👤
Private by design — Each person who installs the app has their own separate local database. Your data is completely independent.
📴
Works offline — Once installed, FinTrack works fully without an internet connection. The app is cached locally.
⚠️
Don't clear site data — Clearing browser site data will delete FinTrack data. Always export a JSON backup first.
📊
Storage limit — IndexedDB typically allows several hundred MB — far more than localStorage. Handles thousands of transactions easily.

📱 Installing the App

🤖
Android (Chrome) — Tap the install button above, or use browser menu → "Add to Home Screen" / "Install App".
🍎
iOS (Safari) — Tap the Share icon (□↑) at the bottom → "Add to Home Screen" → "Add". Must use Safari.
💻
Desktop (Chrome/Edge) — Click the install icon in the address bar, or use the button in this section.
🔄
Moving to another device — Use Export/Import JSON to transfer data. Each device has its own local storage.

🚀 About This App

🌐
Hosted on GitHub Pages — The interface is a static website. Only HTML/CSS/JS is on the server. Your financial data never leaves your device.
🔓
Open source — The code is fully transparent. No hidden tracking, no analytics, no ads.
🛡️
Privacy first — The developer has zero access to your financial data. You own it entirely.
